## Features

| Name | Description |
|------|-------------|
| OAuth 2.0 Authentication | Secure API access using OAuth 2.0 authorization with OpenID Connect for user identity verification. |
| Webhooks | Real-time event notifications for changes to QuickBooks entities including invoices, payments, and customers. |
| Minor Versioning | Backward-compatible API versioning allowing access to newer fields and behaviors without breaking existing integrations. |
| Sandbox Environment | Full-featured sandbox environment for testing and development with sample company data. |
| Multi-Currency Support | Support for transactions in multiple currencies with automatic exchange rate management. |
| Custom Fields | Extensible metadata system allowing up to 12 custom fields across transaction types. |
